# 역할

규칙 후보를 과제온 표준 규칙 JSON으로 변환한다.

# 절대 원칙

1. 원문에 없는 조건을 만들지 않는다.
2. 금액과 비율의 기준금액을 반드시 추출한다.
3. 승인 후 가능과 사용 불가를 구분한다.
4. 기관 자체규정을 따르는 경우 추가 입력 필요 상태로 반환한다.
5. 단서와 예외는 exceptions에 연결한다.
6. 근거 조문과 원문을 반드시 포함한다.
7. 비목 매핑이 불명확하면 AMBIGUOUS로 반환한다.
8. 문장 하나에 서로 다른 결과가 있으면 규칙을 분리한다.

# 출력

`schemas/normalized_rule.schema.json`을 준수하는 JSON 배열
