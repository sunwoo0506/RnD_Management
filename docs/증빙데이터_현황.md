# 비목·항목별 증빙 데이터 현황 (2026-07-22)

집행 화면의 "자동 안내 증빙"을 규정DB 기반으로 바꾸기 위한 사전 조사.
**지금 화면에 뜨는 증빙은 규정DB가 아니라 앱이 넣은 예시 기본값이다.**

## 0. 지금 구조

| 데이터 | 출처 | 화면 |
|---|---|---|
| `category.requiredDocs` | `scripts/convert-regulation-db.mjs`의 `DOCS_BY_CODE` **하드코딩** | 집행 화면 "자동 안내 증빙" ✅ |
| `category.evidenceRules` | 규정DB `regulation_rules.required_documents` | 예산편성 기준 패널 "주의·절차"에만 ⚠️ |
| `allowedItems[].evidence` | 규정DB `expense_allowed_items.evidence_summary` | **어디에도 안 나옴** ❌ |

- 변환 파이프라인은 이미 둘 다 팩까지 싣는다. 막힌 곳은 **화면**과 **데이터 양**이다.
- `AllowedItemList`(`src/App.tsx:464`)가 `evidence` 필드를 렌더링하지 않는다.
- 집행 화면(`src/App.tsx:1340`)은 `documentsFor()` = `requiredDocs`만 쓴다.

## 2. 사업별 상세

### 팁스 (TIPS) — 가장 완비된 원문 · 착수 1순위

- **원문**: `붙임1. 2026년 팁스TIPS 총괄 운영지침 2차 개정안 본문.hwp`
- **위치**: `11.다.1) 비목별 증빙서류` (80쪽) — 추출 텍스트 3,301~3,630행
- **형태**: `세목 | 세세목 | 증빙서류` 3단 표, 약 18~20행
- **덮는 범위**: 인건비(내부/외부/연구근접지원인력), 학생인건비, 연구시설·장비비,
  연구재료비, 연구활동비 9개 세목(지식재산창출·외부전문기술·회의비·출장비·
  소프트웨어활용비·연구실운영비·연구인력지원비·해외연구자유치·종합사업관리비·그 외),
  연구수당, 위탁연구개발비, 국제공동연구개발비, 간접비
- **덤**: 바로 뒤 `11.다.2) 연구개발비 불인정 기준` 표(3,631~3,885행) → `restriction_summary`
- **현재 추출량**: 증빙 규칙 1건, 항목 증빙 2건뿐 (표를 통째로 안 읽은 상태)
- **매핑**: 세세목이 앱의 `allowedItems`와 이름이 거의 같아 1:1로 붙는다

### 예비창업패키지 — 표 있음 · 착수 2순위

- **원문**: `(별첨4) 중소기업창업 지원사업 통합관리지침 제14차.pdf` (55쪽)
- **위치**: **제36조** (16~17쪽)
- **형태**: `비목 | 비목 정의 | 증빙서류` 3단 표
- **특징**: 지급수수료가 세부 항목별로 쪼개져 있다 —
  기술이전비 / 학회·세미나 참가비 / 전시회·박람회 참가비 / 시험·인증비 / 멘토링비 /
  기자재임차비 / 장비수리비 / 사무실임차료 / 운반비 / 보험료 / 보관료 / 회계감사비 /
  해외 액셀러레이터 보육 참가비 / 규제애로 해소 법률컨설팅비
  → 앱의 `allowedItems` 단위와 잘 맞는다
- 인건비·건강검진비·여비(국내/해외)·교육훈련비·광고선전비·기타(비교견적서)도 포함
- 제36조 ②항: 명시되지 않은 증빙은 사업별 세부관리기준을 따르고 추가 요청 가능
- **PDF 텍스트 추출 주의**: 한글 PDF라 어순이 섞여 나온다(`pdfjs-dist`로 추출 가능).
  표 구조는 사람이 확인하며 정리해야 한다

### 국가연구개발비 사용 기준 — 항목별 표 없음

- **원문**: `국가연구개발사업 연구개발비 사용 기준(고시)(제2026-38호).hwpx`
- 증빙 언급 16곳이지만 대부분 보관·정산 절차. **비목별 조항은 8곳**
- 이미 추출됨(5건): 회의비 2, 국외출장 2, 위탁연구개발비 계좌이체증명 1
- **남은 것 3~4건**
  - 연구시설·장비/연구재료 구매 시 **검수확인서** (100만원 이하 검수 간소화 조항 포함)
  - **연구개발부담비** 계좌이체증명 (제63조 ②)
  - **국제공동연구개발비** 계약서 + 계좌이체증명 (인정항목엔 있으나 규칙으로는 없음)
  - **연구혁신비**는 연구비통합관리시스템 입력으로 증명자료 갈음
- 이 고시는 원래 항목별 증빙을 정하는 문서가 아니다 → 여기서 더 나올 게 없다

### 디딤돌 — 원문이 없다

- 관리지침 첫 줄: *"중소기업기술개발 지원사업 관리지침(공통지침)에 따라 운영함을
  원칙으로 하되, 세부과제별 특성에 따라 별도로 정함이 있는 경우 그 내용을 우선"*
- 관리지침 자체에는 증빙 표가 없다 (증빙 언급 2곳, 모두 신청 자격 관련)
- **공통지침 원문을 구해야 한다.** 별도 패키지로 만들고 `basePackId`처럼 계층을 태우는 방식
