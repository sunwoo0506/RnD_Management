# 협약변경 공문 템플릿 (검토용)

변경 관리 탭에서 Word로 내려받을 공문의 문안입니다. **이 문서가 확정되면 그대로 코드에 옮깁니다.**
고치실 곳은 이 파일에서 바로 수정해주세요.

- 작성 규칙 근거: `docs/ai-admin-writer-prompt.md` §6(공문) · §7(- 다 음 -) · §8(협약변경) · §18(붙임)
- 샘플 파일: `docs/samples/협약변경_승인요청_샘플.docx`, `docs/samples/협약변경_통보_샘플.docx`
- 샘플 다시 만들기: `node scripts/make-official-letter-sample.mjs`

## 두 가지 양식

협약변경은 **승인 사항**과 **통보 사항**으로 갈리고, 공문의 성격이 다릅니다.

| | 승인 요청 공문 | 변경 통보 공문 |
|---|---|---|
| 제목 | …**승인 요청** | …**통보** |
| 본문 맺음 | 승인하여 주시기 바랍니다 | 통보하오니 참고하여 주시기 바랍니다 |
| 효력 | 승인 회신 후 집행 | 통보 후 집행 |
| 붙임 | 신청서·대비표·산출내역 | 대비표·산출내역 |

## `{ }` 는 시스템이 채우는 값

| 자리 | 채우는 값 | 출처 |
|---|---|---|
| `{수신기관}` | 전문기관명 | 과제 설정의 주관/전문기관 |
| `{발신기관}` | 기업명 | 과제 설정 |
| `{과제명}` | 과제명 | 과제 설정 |
| `{협약번호}` | 협약번호 | **현재 앱에 없음 — 과제 설정에 추가 필요** |
| `{연구기간}` | 시작일 ~ 종료일 | 과제 설정 |
| `{보내는비목}` `{받는비목}` | 비목명 | 변경 신청 |
| `{이동금액}` `{이동금액한글}` | 금 1,000,000원(금일백만원정) | 변경 신청 |
| `{변경전표}` | 비목별 변경 전·후 대비표 | 편성 + 변경 신청 |
| `{총사업비변동}` | 변동 없음 / 변동 있음 | **확인 안 되면 `[확인 필요]`로 둡니다** |
| `{변경사유}` | 사유 문단 | 변경 신청 (AI 작성 가능) |
| `{작성일}` | 문서 작성일 | 오늘 |
| `{담당자}` `{연락처}` | 담당자 정보 | 담당자·알림 |

# 양식 ① 승인 요청 공문

```
문서번호: {문서번호}
시행일자: {작성일}

수    신: {수신기관}장
참    조: {참조부서}
제    목: 「{과제명}」 사업비 비목 간 변경 승인 요청

1. 귀 기관의 무궁한 발전을 기원합니다.

2. 당사가 수행 중인 「{과제명}」(협약번호: {협약번호}, 연구기간: {연구기간}) 과제와
   관련하여, 연구 수행 과정에서 발생한 사정으로 사업비 비목 간 변경이 필요하여
   아래와 같이 승인을 요청드립니다.

                              - 다  음 -

  가. 변경 개요

      ○ 변경 구분: 사업비 비목 간 변경 (승인 사항)
      ○ 변경 금액: {보내는비목} → {받는비목}  {이동금액한글}
      ○ 총사업비 변동 여부: {총사업비변동}

  나. 변경 전·후 대비

      {변경전표}

  다. 변경 사유

      {변경사유}

  라. 변경 후 사용계획

      {사용계획}

3. 상기 변경사항에 대하여 검토 후 승인하여 주시기 바랍니다.

붙임  1. 사업비 변경 신청서 1부.
      2. 사업비 변경 전·후 대비표 1부.
      3. 변경 반영 사업계획서 1부.  끝.


                        {발신기관}  대표  {대표자명}  (직인)


담당자: {담당자}    연락처: {연락처}
```

## 변경 전·후 대비표 (`{변경전표}` 자리)

금액이 바뀐 비목만 넣고, 마지막에 합계를 둡니다. 바뀌지 않은 비목까지 다 넣으면
검토자가 어디가 달라졌는지 찾아야 합니다.

| 비목 | 변경 전 | 변경 후 | 증감 |
|---|---:|---:|---:|
| 인건비 | 60,000,000원 | 59,000,000원 | △1,000,000원 |
| 연구재료비 | 20,000,000원 | 21,000,000원 | 1,000,000원 |
| **합계** | **100,000,000원** | **100,000,000원** | **0원** |

- 감액은 `△`로 표시합니다 (행정문서 관행).
- 합계 행의 증감이 0원이 아니면 총사업비가 바뀐 것이므로 `{총사업비변동}`도 그에 맞춰야 합니다.

## 확인이 필요한 것

구현 전에 정해주셔야 하는 항목입니다.

1. **협약번호** — 지금 앱에 없는 값입니다. 과제 설정에 칸을 추가할까요?
2. **문서번호** — 기관마다 채번 방식이 다릅니다. 빈칸으로 두고 직접 적게 할까요,
   아니면 `{기업약칭}-{연도}-{일련번호}` 같은 형식을 만들까요?
3. **대표자명** — 지금은 담당자만 있고 대표자명이 따로 없습니다. 추가할까요?
4. **참조부서** — 대부분 비워두는데, 칸 자체를 없앨까요?
5. **사용계획** — 승인 공문에만 있는 항목입니다. AI 사유 작성에서 함께 뽑을까요,
   아니면 별도 입력칸을 둘까요?
6. **직인** — Word에서 `(직인)` 글자로 두고 출력 후 날인하는 방식이면 충분한지.
