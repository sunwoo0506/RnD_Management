# 과제온 예비창업패키지 규정 DB

MVP 산출물 규격(`docs/gwayeon_guideline_extraction_framework/04_mvp_output_spec.md`)에 맞춰 만든 패키지입니다.

## 기준

- 문서: 2026 예비창업패키지 예비창업자 수정 모집공고(2차)
- 공고: 중소벤처기업부 공고 제2026-207호
- 공고일: 2026-03-24
- 협약기간: 협약 시작일로부터 8개월 이내 (2026년 6월 ~ 2027년 1월 예정)

규정 계층 — **공고(2026-207호) > 세부관리기준(2025.7.2. 시행) > 통합관리지침 제14차(2025.12.23. 전부개정) > 운영요령 고시 제2024-101호**.
공고가 정한 사항이 우선하고, 공고에 없는 집행 세부기준은 통합관리지침·세부관리기준을 따릅니다.
각 규칙의 `source_article`에 어느 문서에서 왔는지 표시해 뒀습니다.

## 이 사업의 특징

**비목 간 비율 제한이 없습니다.** QnA 사업비 3번이 "비목 간의 비율은 정해져 있지 않으므로 사업추진을
위해 필요한 항목에 계상하시기 바랍니다"라고 명시합니다. 그래서 규정 팩의 `hasRatioLimits`가 `false`이고
편성 화면은 상한 대신 금지·주의 중심으로 표시됩니다. 상위 규정에 비율이 몇 개 나오지만(외주용역 선금
50% 이하, 현물 참여율 100% 이내) 이는 집행·참여율 제한이지 비목 편성 금액의 상한이 아니라서
`expense_limit_rules`가 아닌 `regulation_rules`에 담았습니다.

**간접비가 없습니다.** 사업화 자금 9개 비목만 있는 체계라 다른 R&D 사업의 직접비/간접비 구조와 다릅니다.

**재원은 정부지원사업비 100%입니다.** 자기부담사업비가 없어 현물 계상 개념도 없습니다.

**집행 세부기준은 공고가 아니라 상위 규정에 있습니다.** 공고문은 "선정자는 「중소기업창업 지원사업
운영요령」, 「중소기업창업 지원사업 통합관리지침」, 「동 사업 세부관리기준」을 준수해야 한다"고만 적고
기준 자체는 담지 않습니다. 그래서 이 패키지는 세 문서를 함께 추출해 규칙을 채웠습니다. 공고와 QnA만으로는
11건이던 규칙이 42건이 됐습니다.

상위 규정에서 온 대표적인 기준:

- **대표자 인건비는 지급 불가**, 현물로만 계상 (통합관리지침 제41조제1항) — 공고의 배우자·직계존비속 금지보다 넓습니다
- **2천만원 초과 외주용역은 사전심의**, 선금은 계약금액의 50% 이하 (제38조제3항·제5항)
- **PC·사무용복합기는 과제와 직접 관련될 때만**, 범용 사무용 소프트웨어는 사전검토 (제39조)
- **취득자산은 처분 제한 및 5년간 관리** (제39조제6항·제7항)
- **여비는 증빙 가능한 대중교통비만**, 국외 출장은 사전 보고 (제43조·세부관리기준 제22조)
- **개인과의 거래 불가** (멘토링비 제외), 집행 전 주관기관 사전검토 (제47조)

사전승인·사전검토가 필요한 규칙이 5건이라, 편성 단계에서 미리 알려주는 것이 이 사업에서는 특히 유용합니다.
