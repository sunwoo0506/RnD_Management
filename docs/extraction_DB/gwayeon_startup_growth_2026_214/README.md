# 과제온 창업성장기술개발사업(디딤돌) 규정 DB

MVP 산출물 규격(`docs/gwayeon_guideline_extraction_framework/04_mvp_output_spec.md`)에 맞춰 만든 패키지입니다.
대상은 **디딤돌 도약(전략기술 R&D)** 과제입니다.

## 기준

- 문서: 2026년도 창업성장기술개발사업(디딤돌) 제3차 시행계획 공고
- 공고: 중소벤처기업부 공고 제2026-214호
- 시행: 2026-06-01
- 지원규모: 총 100억원, 150개 내외 / 과제당 정부지원연구개발비 최대 2억원(연 1억원), 최대 1년 6개월

규정 계층 — **공고(2026-214호) > 디딤돌 관리지침 > 공통지침·운영요령 > 「국가연구개발사업 연구개발비 사용 기준」**.
연구개발비 산정·사용의 일반 기준은 국가연구개발비 사용 기준을 따르므로, 이 패키지에는 **디딤돌 사업이 별도로
정하거나 명시한 규칙만** 담았습니다. 비목 체계도 국가연구개발사업 비목을 그대로 쓰기 때문에 코드를 NRD
패키지(`gwayeon_regulation_db_2026_38_final`)와 동일하게 맞춰, 앱에서 공통 기준을 이름으로 찾아 붙일 수 있습니다.

## 이 사업의 특징

**사업화 지원비 200만원이 필수 계상 항목입니다.** 도약 과제는 전문기관이 지정한 업무지원기관의 특화 프로그램
비용을 연구활동비 내 외부전문기술 활용비로 반드시 계상해야 하고, 1차년도 종료 전까지 업무지원기관에 지급해야
하며, **계상·지급하지 않으면 협약이 해약될 수 있습니다**(관리지침 사.3) 다)). 근거가 세 곳(붙임2-5 ·
관리지침 바.2) · 관리지침 사.3))에 흩어져 있어 규칙 3건(`DD-ACT-001~003`)으로 나눠 담았습니다.

**기존 인력 인건비는 현물계상이 원칙입니다.** 영리기관 소속 참여연구자는 창업 7년 이내 기업만 현금 계상이
가능하고, 「신산업 창업 분야에 관한 규정」(중기부고시 제2024-2호) 대상 중소기업은 창업 10년 이내까지
허용됩니다. 신규채용 인건비는 중소기업이면 현금 계상이 가능합니다.

**연구실운영비는 평가위원회 승인이 필요합니다.** 영리기관이 사무용 기기·사무용 소프트웨어·연구실 환경유지기기를
계상하려면 활용·계획(별첨 양식)을 제출해 승인받아야 하고, **사무용품은 제외**됩니다.

**재원은 정부지원 75% / 기관부담 25%**이고, 기관부담연구개발비(현금+현물) 중 **10% 이상은 현금**이어야
합니다. 공고 예시 기준으로는 총 연구개발비 266,750천원 중 정부지원 200,000천원, 기관부담 현금 6,750천원·
현물 60,000천원입니다.

**연구수당은 증액 계상이 불가합니다.** 협약 체결 당시 계상한 금액을 넘겨 계상할 수 없습니다.

공통 규칙 6건(RCMS 집행, 협약 변경 절차 등)은 모든 비목에 표시됩니다.

상한 4건은 비목이 아니라 **세부항목이나 조정된 산정 기준**을 기준으로 삼습니다 — 외부 전문기술 활용비는
연구활동비 전체가 아닌 세목 기준, 위탁은 "직접비에서 위탁·국제공동·연구개발부담비를 제외한 금액" 기준,
간접비는 "직접비에서 미지급인건비·현물·국제공동·위탁을 제외한 금액" 기준입니다. 그래서 화면에서는 계산식과
기준을 문구로 보여주고 자동 계산은 하지 않습니다.

참조 비목(`referenceCategories`) 2건 — 외부 전문기술 활용비, 연구실운영비. 사용자가 AI 추출 팩처럼 다른
이름의 비목을 쓰더라도 이름으로 기준을 찾아 붙이기 위한 것입니다.
