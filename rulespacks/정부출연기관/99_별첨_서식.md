# 별첨 — 정부출연기관 관련 별표·별지 서식

> 국가연구개발사업 연구개발비 사용 기준(개정안) 중 정부출연기관 조문이 참조하는 별표·서식.
> 양식은 원문 기준으로 정리한 것이며, 실제 제출 시 최신 고시 서식을 확인하세요.

---

## 별표 1 — 기본사업연구개발비계상기준 (제44조 제2항 관련)

정부출연기관 기본사업의 항목별 계상 기준입니다. "출연금인건비 / 주요사업비 / 출연금경상비 / 출연금건축비"로 계상처가 나뉩니다.

| 항목 | 계상기준 |
|------|----------|
| 1. 인건비 | 정규직 연구자·연구근접지원인력 → **출연금인건비**. 비정규직 → 주요사업비. 미소속 연구자 → 주요사업비(단, 소속기관 지급 인건비 상당액은 출연금 계상 불가·현물만 가능) |
| 2. 학생인건비 | 주요사업비 |
| 3. 연구시설·장비비 | 주요사업비·출연금건축비 |
| 4. 연구재료비 | 주요사업비 |
| 5. 연구활동비 | 주요사업비 |
| 6. 연구수당 | 주요사업비 |
| 7. 위탁연구개발비 | 주요사업비 |
| 8. 국제공동연구개발비 | 주요사업비 |
| 9. 연구개발부담비 | 주요사업비·출연금인건비 |
| 10. 인력지원비 | 연구지원인력 인건비 → 출연금인건비. 연구개발능률성과급·제15조제4호 급여 → 출연금·주요사업비 계상 불가 |

> 주: 기초과학연구원 등 일부 기관·항목은 특례가 있음. 원문 별표 1 전문 참조.

## 별지 제1호·제2호서식 — 학생연구자 연구참여확약서

**참조 조문:** 제40조 제2항 (학생인건비계상률 관리)

학생인건비를 계상하는 정부출연기관은 학생연구자별로 연구참여확약서를 학기(6개월) 또는 학년(12개월) 단위로 작성해야 합니다. (근로계약 체결 시 생략 가능)

- 별지 제1호: 학생인건비통합관리 **연구책임자계정**용
- 별지 제2호: (대응 서식)

### 1. 학생연구자 인적사항
- 성명 / 학번 / 생년월일 / 소속(학과명)
- 과정 및 학기 / 국가연구자번호 / 학적상태(재학·휴학·수료등록)
- 주소 / 연락처(휴대전화·E-mail)

### 2. 연구참여정보
- 연구참여기간: 20 . . ~ 20 . .
- 담당업무(역할): 연구수행내용·역할 구체적 작성 (연구 무관 업무 기재 불가)
- 계정명
- **월 학생인건비 지급액 및 지급률** (세금 포함)

> 확약서는 학생연구자 본인·연구책임자(계정책임자)가 서명하여 체결하며, 협의를 거친 경우에만 변경 가능(제40조 제3항).
