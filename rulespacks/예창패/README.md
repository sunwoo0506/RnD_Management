# 예비창업패키지 편 — 공고 기준 규정 정리

2026 예비창업패키지 모집공고(2026-207호) 및 주요 질의응답에서 추출한 규정 데이터입니다. 앞서 설계한 "공고 업로드 → 규정 추출 → 우선순위 매칭" 흐름의 실제 적용 사례입니다.

## ⚠️ R&D 지침과 다른 규정 체계

예비창업패키지는 **창업사업화 지원사업**으로, 국가연구개발사업 연구개발비 사용 기준(R&D)과 규정 체계가 다릅니다.

| | R&D 과제 | 예비창업패키지 |
|---|---|---|
| 근거 지침 | 국가연구개발 연구비 사용 기준 | 중소기업창업 지원사업 통합관리지침 |
| 비목 비율 | 상한 있음(연구수당 20% 등) | **비율 제한 없음** (QnA 사업비 3번) |
| 규정 핵심 | 비율 준수 | **비목별 집행 가능/불가** |
| 사업비 구성 | 정부+기관부담 등 | 정부지원 100% |
| 참여율 | 100~130% | 해당 개념 없음 |

→ 매칭 엔진은 사업 유형을 먼저 식별해서 어떤 규정 체계를 적용할지 결정해야 합니다.

## 스키마 (R&D와 다른 점)

R&D는 `ratio_rule`(비율)이 중심이지만, 예비창업패키지는:
- `expense_category`: 비목별 집행 가능 여부(allowed) + 정의
- `warning`: 집행 금지 규칙 (친인척 인건비, 사무가구, 소급, 잔여 반납)
- `info`: 참고 규정 (비율 없음, 인건비 산정 기준, 총사업비 구성)

각 규칙에 `matchLevel`(notice/notice_guideline)과 `evidence_ref`(공고·QnA 근거)를 붙여 출처를 추적합니다.

## 집행 비목 (공고 기준, 9종 모두 허용)

재료비 · 외주용역비 · 기계장치(공구·기구·비품·SW) · 특허권 등 무형자산 취득비 · 인건비 · 지급수수료 · 여비 · 교육훈련비 · 광고선전비

## 집행 불가 · 주의 (공고 + QnA)

- 대표자 배우자·직계존비속 인건비 (공고 비목 정의 + QnA 사업비 7번)
- 사무용 집기·가구·기구 — 책상·서랍 등 (QnA 사업비 4번)
- 선정 전 집행분 소급 청구 (QnA 사업비 2번)
- 협약 종료 후 잔여 사업비 사용 → 전액 국고 반납 (QnA 사업비 6번)

## 검증 상태

공고·QnA 원문에서 추출했으나 공식 검증 전(example)입니다. 실제 협약서 및 최신 통합관리지침(제14차)과 대조 후 확정하세요.
