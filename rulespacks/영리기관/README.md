# 영리기관 편 — 연구개발비 사용 기준 정리 폴더

「국가연구개발사업 연구개발비 사용 기준」(개정안)을 **영리기관** 기준으로, AI(RAG/챗봇) 활용과 화면 가이드용으로 정리한 자료 모음입니다. 처음 정리했던 연구비 관리 지침 폴더와 동일한 포맷(장·절 단위 분할 + `> ⚠️ 이슈 배너` + index + 별첨)을 씁니다.

## 폴더 구성

| 파일 | 용도 |
|------|------|
| `00_index.md` | 인덱스 — **항상 첨부**. 챕터 목록, 영리기관 핵심 기준 요약, 별첨 목록 |
| `01_공통_사용용도.md` | 제10조 회의비 등 기관 무관 사용용도 |
| `02_공통_수당_위탁.md` | 제26·26의2·27조 연구수당·보안수당·위탁비 (기관 무관) |
| `03_영리기관_계상기준.md` | 제6절 제60~69조 영리기관 특례 (본문 핵심) |
| `99_별첨_서식.md` | 별지 제3호(현금 인건비 관리)·제4호(연구실운영비 계획) 서식 |
| `rd_cost_chunks_forprofit.jsonl` | RAG용 조문 청크 (structured 필드 포함) |

## 사용법

**AI 챗봇/RAG:**
- `rd_cost_chunks_forprofit.jsonl`의 각 청크 `embed_text`를 임베딩
- `is_issue`·`issue_category`·`org_type`으로 사전 필터
- 답변 근거로 `content`, 힌트로 `common_questions` 활용

**화면 가이드:**
- 각 청크의 `structured` 블록에서 비율 표(ratio_rule)·증빙 체크리스트(evidence)·경고(warning) 파생
- 각 규칙에 `source_ref`가 붙어 있어 "왜 이 규정인지" 조문 근거 제시 가능
