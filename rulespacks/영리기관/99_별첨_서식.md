## 별지 제3호서식 — 영리기관의 현금 계상 인건비 관리 계획 및 현황

**참조 조문:** 제65조 제6항 (현금 계상 인건비 구분 관리)

### 1. 연구개발과제 현황
- 사업명 / 연구개발과제명 / 연구개발기관명
- 연구책임자 (소속·성명)
- 연구개발기간: 전체 / 1단계 / n단계 (해당 시)

### 2. 영리기관의 연구자 인건비 현금 계상 내역 (단위: 천원)

| 구분 | A. 신규채용 참여연구자 현금 인건비 | B. 전문연구사업자 소속 참여연구자 현금 인건비 | C. 장관 인정 참여연구자 현금 인건비 | 합계 (A+B+C) |
|------|:---:|:---:|:---:|:---:|
| | 원래계획 / 변경후 | 원래계획 / 변경후 | 원래계획 / 변경후 | 원래계획 / 변경후 |
| 전체기간 | | | | |
| 1단계 1년차 | | | | |
| 1단계 2년차 | | | | |
| n단계 m년차 | | | | |

**작성 주석**
- '원래계획 금액': 협약 체결 시 계획 금액. 변경된 경우 현행 계획 금액 입력.
- '변경 후 금액': 협약 변경으로 변경하려는 금액 입력.
- A: 제65조 제4항 제1호. 신규 채용 참여연구자(채용일~공고일 6개월 이내 포함).
- B: 제65조 제4항 제2호. 전문연구사업자로 신고한 기관 소속 참여연구자.
- C: 제65조 제4항 제3~6호. 장관이 현금 계상 필요를 인정한 참여연구자.
- 단계 구성에 따라 행 추가·삭제 가능.

*서명: 연구책임자 (인) / 연구개발기관장 (직인) / OOO의 장 귀하*

## 별지 제4호서식 — 영리기관의 연구실운영비 활용·관리 계획

**참조 조문:** 제68조 제4항 (연구실운영비 계상 요건)

### 1. 연구개발과제 현황
- 사업명 / 연구개발과제명 / 연구개발기관명
- 연구책임자 (소속·성명)
- 연구개발기간: 전체 / 1단계 / n단계 (해당 시)

### 2. 영리기관의 연구실운영비 (금액, 단위: 천원)

| 구분 | 소모성 비용 | A. 사무용기기·SW 비용 | B. 냉난방·환경유지 기기·비품 비용 | 합계 (C=A+B) |
|------|:---:|:---:|:---:|:---:|
| 전체기간 | (사용불가) | | | |
| 1단계 | (사용불가) | | | |
| n단계 | (사용불가) | | | |

**작성 주석**
- 최초 협약 체결 시 계획 금액 입력. 협약 후 수정 필요 시 중앙행정기관 장의 사전승인 후 협약변경.
- 사무용품비는 이 계획과 별도로 사용 가능.
- A: 사무용기기·소프트웨어의 구입·설치·임차·사용대차 비용.
- B: 냉난방·환경유지에 필요한 기기·비품의 구입·유지 비용.
- '소모성 비용'은 계상 불가 항목.
